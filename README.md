Payroll Calculator Extension

Version: 1.0.0
Updated: November 29, 2025

Overview

Payroll Calculator is a browser extension that helps users quickly calculate and track their pay for multiple shifts and pay periods.

Key Features:

  Calculates gross pay, tax, NI, and take-home pay per shift and per period.

  Supports holiday pay, extra deductions, and tax-free allowances.

  Summarizes multiple pay periods with expandable detailed views.

  Local storage of hourly rates, holiday rates, and extra values for convenience.

  Responsive UI with toggleable sections for clean organization.

  All code is original, built using JavaScript, HTML, and CSS.

Usage

  Enter hourly rate, holiday rate, and hours worked.

  Add extra deductions or tax-free allowances if applicable.

  View per-shift and per-period calculations in a clear, expandable format.

  Optionally clear stored pay data for a specific month.

Privacy

  All calculations and data are stored locally.

  No personal data is transmitted or collected.
